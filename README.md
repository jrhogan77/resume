# Jason Hogan — Resume & Cover Letter

Professional documents authored entirely in HTML and CSS — no frameworks, no build tools, no dependencies beyond a Google Fonts import. Designed for browser rendering and PDF export via browser print (`Ctrl+P` → Save as PDF).

## Design System

| Token | Value | Role |
|-------|-------|------|
| `--navy` | `#1A1F2E` | Sidebar background, primary text |
| `--orange` | `#D4622A` | Accent color, section markers |
| `--orange-lt` | `#F0845A` | Sidebar labels, highlights |
| `--orange-bg` | `#FDF3EE` | Callout block backgrounds |
| `--sidebar-w` | `260px` | Resume sidebar width |

**Fonts:** Inter (Google Fonts) — body copy at weights 300–700  
**Display font:** Playfair Display — name and headings  
**Layout:** CSS Flexbox — fixed sidebar + fluid main column  
**Print:** `@page { size: letter; margin: 0; }` — full-bleed zero-margin PDF output
